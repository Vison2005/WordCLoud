# WordCLoud
这是一个用于生成词云图的软件，当前版本（WordCloud.py）基于PyQt5进行界面编辑，左侧缩略图选择框可在thumbnail文件夹存入背景透明的图片用于生成词云图，整个窗口均允许拖入图片，.txt文件，.ttf字体，可自动生成，点击下方更新词云图可更换新的样式，右侧选择颜色可选择字体主要颜色，两个框内可以添加用于生成的颜色，背景颜色仅能选择单色。选择颜色可以保存用户配置，点击add to costom即可。也可从屏幕上选择对应颜色，内置取色器。右侧文本选择框内可以加载位于text文件夹中的.txt文件。
实现原理：利用jieba库进行分词，生成对应的"文件名count.txt"，用wordcloud和imageio绘制对应的词云图。
